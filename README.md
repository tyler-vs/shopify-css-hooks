# shopify-css-hooks
Snippet for CSS hooks

## Installation

1. Add [`css-hooks.liquid`](./css-hooks.liquid) to the Snippets/ folder.
2. Go to `theme.liquid` add `{% render 'css-hooks' %}` in the `<body>` elements class attribute. (e.g. `<body class="{% render 'css-hooks' %}">`)
